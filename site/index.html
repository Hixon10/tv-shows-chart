Table macros

Use these in FROM. They return tables; combine them with JOIN or IN (SELECT …).

MacroReturnsExample
scrape_date_between(d1, d2) show_ids active in the window SELECT s.* FROM shows s JOIN scrape_date_between('2026-05-01','2026-05-30') USING (show_id);
hot_in(d1, d2) shows + in-window aggregates, sorted by avg rank SELECT * FROM hot_in('2026-05-25','2026-05-30') LIMIT 20;
rank_history(show_id) per-day rank, score, votes per source SELECT * FROM rank_history((SELECT show_id FROM shows LIMIT 1));
find_show(q) shows whose assumed/Russian title contains q (case-insensitive) SELECT * FROM find_show('boys');
